# Memory Bank — Project Brief

## What is FundabilityOS?

FundabilityOS is a AI fundability dianogstic platform built by NextBlaze Asia that bridges the gap between fundraising startups and  investors. It provides an objective, AI-powered "fundability score" (0–100) that tells founders exactly where they stand before approaching investors.

## Core Problem

Founders waste months preparing for investor meetings without knowing their real weaknesses. Investors waste time in first meetings with startups that are not ready. FundabilityOS solves both sides.

## Vision

Become the **standard pre-funding diagnostic** for startups across Southeast Asia and beyond — the Bloomberg Terminal for early-stage fundraising.

## Primary Goals

1. Give every startup founder a free, instant fundability score.
2. Give investors a curated, pre-qualified deal flow (score ≥ 75 only).
3. Use investor feedback to continuously improve scoring accuracy.
4. Monetise the investor side ($99–$499/month) while keeping startups free.

## Success Criteria

- 1,000 startups assessed in first 90 days.
- 50 paying investors on the platform in first 6 months.
- Score accuracy validated: investor "invest" decisions correlate with high scores.
- Email delivery working for all assessment completions.
- Zero security incidents (hardcoded secrets, RLS bypass, data leaks).

## Out of Scope (Current Phase)

- Mobile app
- Real-time investor-founder messaging
- Cap table management
- Full due diligence workflow
